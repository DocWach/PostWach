COALITION-Q01: Data Classification Constraints
What data classification constraints and release policies limit mission-critical information sharing with coalition partners, and how do these constraints affect joint mission effectiveness?
HIGH
Wargame Design

Type: Matrix Game with classification layers
Format: 2-day multinational tabletop
Participants: 15-25 (multi-nation representation)

Neural AI Contribution

Learn historical patterns of classification friction; predict information bottlenecks; generate realistic data-sharing scenarios

Symbolic AI Contribution

Encode classification rules (STANAG 4774); enforce release constraints; trace information flow decisions; explain policy violations

Measures

MoE: Joint mission effectiveness; information availability
MoP: Data release latency; classification overhead

Acquisition Relevance

C4ISR system requirements; cross-domain solutions

DOTMLPFI

Interoperability, Policy, Leadership

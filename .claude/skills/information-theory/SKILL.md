# Information Theory Skill

## Invocation
```
/information-theory [subcommand] [arguments]
```

## Subcommands
- `/information-theory entropy <distribution>` - Compute entropy
- `/information-theory capacity <channel>` - Channel capacity
- `/information-theory coding <code>` - Analyze error-correcting code
- `/information-theory rate-distortion <source>` - Rate-distortion analysis

## References
- Cover & Thomas (2006). Elements of Information Theory

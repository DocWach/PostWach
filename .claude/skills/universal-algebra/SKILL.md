# Universal Algebra Skill

## Overview

This skill provides methodologies for studying algebraic structures abstractly—signatures, operations, varieties, clones, and Maltsev conditions. It captures common patterns across groups, rings, lattices, and other algebras.

## When to Use

- Analyzing algebraic structures abstractly
- Working with varieties and HSP
- Clone and term operation analysis
- Maltsev conditions for congruence properties
- Subdirect representation

---

## Algebraic Structures

```
SIGNATURES AND ALGEBRAS
═══════════════════════════════════════════════════════════════

SIGNATURE
─────────────────────────────────────────
σ = (F, ar) where F = operation symbols, ar = arities

Examples:
  Groups: {·, ⁻¹, e} with arities (2, 1, 0)
  Lattices: {∨, ∧} with arities (2, 2)

ALGEBRA
─────────────────────────────────────────
A = (A, F^A) where A is carrier, F^A interprets operations.

TERMS
─────────────────────────────────────────
Built from variables and operations.
Term t defines operation t^A: A^n → A.

IDENTITIES
─────────────────────────────────────────
A ⊨ s ≈ t means s^A = t^A as functions.
```

## Subalgebras, Homomorphisms, Products

```
BASIC CONSTRUCTIONS
═══════════════════════════════════════════════════════════════

SUBALGEBRA
─────────────────────────────────────────
B ≤ A if B closed under all operations.
⟨X⟩_A = subalgebra generated by X.

HOMOMORPHISM
─────────────────────────────────────────
h: A → B preserving operations:
  h(f^A(a₁,...,aₙ)) = f^B(h(a₁),...,h(aₙ))

CONGRUENCE
─────────────────────────────────────────
Equivalence compatible with operations.
A/θ = quotient algebra.
ker(h) is congruence, A/ker(h) ≅ h(A).

PRODUCTS
─────────────────────────────────────────
∏Aᵢ with pointwise operations.
Subdirect: projections surjective.
```

## Variety Theory

```
HSP THEOREM
═══════════════════════════════════════════════════════════════

CLASS OPERATORS
─────────────────────────────────────────
H(K) = homomorphic images
S(K) = subalgebras
P(K) = products

BIRKHOFF'S THEOREM
─────────────────────────────────────────
K is variety iff K = Mod(Σ) for some identities Σ.

Varieties = HSP-closed classes = Equational classes.

FREE ALGEBRAS
─────────────────────────────────────────
F_V(X) = free algebra in V over generators X.
Universal property: Maps X → A extend uniquely to homomorphisms.

SUBDIRECT REPRESENTATION
═══════════════════════════════════════════════════════════════

Every algebra is subdirect product of subdirectly irreducibles.
A is SI iff Con(A) has unique minimal element > 0.
```

## Clone Theory

```
CLONES
═══════════════════════════════════════════════════════════════

DEFINITION
─────────────────────────────────────────
Clone: Set of operations containing projections, closed under composition.
Clo(A) = clone of all term operations of A.

GALOIS CONNECTION
─────────────────────────────────────────
Pol(R) = operations preserving relation R
Inv(C) = relations preserved by clone C

Pol-Inv connection:
  Clone C = Pol(Inv(C))
  Relational clone = Inv(Pol(R))
```

## Maltsev Conditions

```
CONGRUENCE PROPERTIES
═══════════════════════════════════════════════════════════════

MALTSEV TERM
─────────────────────────────────────────
V is congruence permutable iff has term p(x,y,z):
  p(x,x,y) = y, p(x,y,y) = x

Examples: Groups (p = xy⁻¹z), quasigroups.

JÓNSSON TERMS
─────────────────────────────────────────
V is congruence distributive iff has Jónsson terms d₀,...,dₙ:
  d₀(x,y,z) = x, dₙ(x,y,z) = z
  dᵢ(x,y,x) = x, plus alternating conditions.

Example: Lattices.

HIERARCHY
─────────────────────────────────────────
Permutable → Modular → n-permutable → ...
Distributive → Modular

JÓNSSON'S LEMMA
─────────────────────────────────────────
In congruence distributive variety V = HSP(K):
SI algebras in V belong to HS(K).
```

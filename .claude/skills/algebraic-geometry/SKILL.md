# Algebraic Geometry Skill

## Overview

This skill provides methodology for algebraic geometry, including varieties, schemes, sheaves, cohomology, and intersection theory. It coordinates with the algebraic-geometer agent.

## Invocation

```
/algebraic-geometry [subcommand] [arguments]
```

## Subcommands

### `/algebraic-geometry variety <equations>`
Analyze an algebraic variety defined by polynomial equations.

### `/algebraic-geometry scheme <spec>`
Analyze a scheme (affine, projective, or general).

### `/algebraic-geometry sheaf <sheaf>`
Compute sheaf cohomology and related invariants.

### `/algebraic-geometry divisor <divisor>`
Analyze divisors and line bundles.

### `/algebraic-geometry morphism <map>`
Analyze morphisms between varieties/schemes.

### `/algebraic-geometry intersection <cycles>`
Compute intersection numbers and products.

## Methodology

### Variety Analysis

```
VARIETY ANALYSIS
═══════════════════════════════════════════════════════════════

STEP 1: DEFINE AND EMBED
─────────────────────────────────────────
- Specify defining equations
- Identify ambient space (affine/projective)
- Check irreducibility

STEP 2: COMPUTE INVARIANTS
─────────────────────────────────────────
- Dimension
- Degree (in projective space)
- Singular locus

STEP 3: ANALYZE STRUCTURE
─────────────────────────────────────────
- Coordinate ring / function field
- Picard group
- Canonical class
```

### Sheaf Cohomology

```
SHEAF COHOMOLOGY
═══════════════════════════════════════════════════════════════

COMPUTATION METHODS
─────────────────────────────────────────
1. Čech cohomology (for explicit open covers)
2. Long exact sequence from short exact sequences
3. Serre vanishing for high twists
4. Kodaira vanishing for ample bundles

STANDARD SEQUENCES
─────────────────────────────────────────
Ideal sheaf sequence:
0 → I_Z → O_X → O_Z → 0

Euler sequence on ℙⁿ:
0 → O(-1) → O^{n+1} → T_ℙⁿ(-1) → 0

Normal bundle sequence:
0 → T_Z → T_X|_Z → N_{Z/X} → 0
```

### Riemann-Roch Application

```
RIEMANN-ROCH
═══════════════════════════════════════════════════════════════

FOR CURVES
─────────────────────────────────────────
ℓ(D) - ℓ(K-D) = deg(D) + 1 - g

FOR SURFACES
─────────────────────────────────────────
χ(O_S(D)) = χ(O_S) + ½D·(D-K)

GENERAL (Hirzebruch-Riemann-Roch)
─────────────────────────────────────────
χ(X, E) = ∫_X ch(E) · td(X)
```

## Examples

### Example: Cubic Surface

```
/algebraic-geometry variety "x³ + y³ + z³ + w³ = 0 in ℙ³"

VARIETY ANALYSIS: Cubic Surface
═══════════════════════════════════════════════════════════════

DEFINITION
─────────────────────────────────────────
V: x³ + y³ + z³ + w³ = 0 ⊂ ℙ³
Fermat cubic surface

DIMENSION & DEGREE
─────────────────────────────────────────
dim = 2 (surface)
deg = 3 (cubic)

SINGULARITIES
─────────────────────────────────────────
Smooth (Jacobian criterion: no common zeros of partials)

INVARIANTS
─────────────────────────────────────────
Pic(S) = ℤ^7 (generated by 27 lines)
K_S = O_S(-1) (anticanonical = hyperplane)
χ(O_S) = 1
p_g = q = 0 (rational surface)

SPECIAL PROPERTIES
─────────────────────────────────────────
- Contains exactly 27 lines
- Rational (birational to ℙ²)
- del Pezzo surface of degree 3
- Blow-up of ℙ² at 6 points in general position
```

### Example: Divisor Computation

```
/algebraic-geometry divisor "degree 3 on genus 2 curve"

DIVISOR ANALYSIS
═══════════════════════════════════════════════════════════════

SETUP
─────────────────────────────────────────
C = smooth curve of genus g = 2
D = effective divisor of degree 3

RIEMANN-ROCH APPLICATION
─────────────────────────────────────────
ℓ(D) - ℓ(K-D) = deg(D) + 1 - g = 3 + 1 - 2 = 2

Since deg(K-D) = 2g - 2 - 3 = 1 ≥ 0, may have ℓ(K-D) > 0

CASE ANALYSIS
─────────────────────────────────────────
If D is general: ℓ(K-D) = 0, so ℓ(D) = 2
If D is special: ℓ(K-D) ≥ 1, so ℓ(D) ≥ 3

GEOMETRIC INTERPRETATION
─────────────────────────────────────────
|D| gives map C → ℙ¹ (degree 3 cover)
General divisor: simple map
Special divisor: factors through lower degree
```

# Approximation Theory Skill

## Invocation
```
/approximation-theory [subcommand] [arguments]
```

## Subcommands
- `/approximation-theory polynomial <function>` - Polynomial approximation
- `/approximation-theory best <function>` - Best approximation
- `/approximation-theory spline <data>` - Spline approximation
- `/approximation-theory wavelet <function>` - Wavelet analysis
- `/approximation-theory rate <method>` - Convergence rates

## References
- DeVore & Lorentz (1993). Constructive Approximation

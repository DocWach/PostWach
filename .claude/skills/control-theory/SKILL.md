# Control Theory Skill

## Invocation
```
/control-theory [subcommand] [arguments]
```

## Subcommands
- `/control-theory state-space <system>` - State space analysis
- `/control-theory controllability <system>` - Check controllability
- `/control-theory stability <system>` - Stability analysis
- `/control-theory lqr <system>` - LQR design
- `/control-theory lyapunov <system>` - Lyapunov analysis

## References
- Khalil (2002). Nonlinear Systems

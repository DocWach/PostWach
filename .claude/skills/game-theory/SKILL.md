# Game Theory Skill

## Invocation
```
/game-theory [subcommand] [arguments]
```

## Subcommands
- `/game-theory nash <game>` - Find Nash equilibria
- `/game-theory dominant <game>` - Find dominant strategies
- `/game-theory cooperative <game>` - Analyze cooperative game
- `/game-theory mechanism <objective>` - Design mechanism

## References
- Osborne & Rubinstein (1994). A Course in Game Theory

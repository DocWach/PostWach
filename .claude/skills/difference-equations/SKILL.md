# Difference Equations Skill

## Invocation
```
/difference-equations [subcommand] [arguments]
```

## Subcommands
- `/difference-equations solve <equation>` - Solve difference equation
- `/difference-equations stability <system>` - Analyze stability
- `/difference-equations ztransform <sequence>` - Apply Z-transform
- `/difference-equations functional <equation>` - Solve functional equation

## References
- Elaydi (2005). An Introduction to Difference Equations

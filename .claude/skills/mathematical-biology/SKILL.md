# Mathematical Biology Skill

## Invocation
```
/mathematical-biology [subcommand] [arguments]
```

## Subcommands
- `/mathematical-biology population <model>` - Population dynamics
- `/mathematical-biology epidemic <model>` - Epidemiological models
- `/mathematical-biology reaction-diffusion <system>` - Pattern formation
- `/mathematical-biology stability <equilibrium>` - Stability analysis

## References
- Murray (2002). Mathematical Biology I & II

# Integral Equations Skill

## Invocation
```
/integral-equations [subcommand] [arguments]
```

## Subcommands
- `/integral-equations fredholm <equation>` - Solve Fredholm equation
- `/integral-equations volterra <equation>` - Solve Volterra equation
- `/integral-equations spectral <kernel>` - Spectral analysis
- `/integral-equations numerical <equation>` - Numerical methods

## References
- Kress (2014). Linear Integral Equations

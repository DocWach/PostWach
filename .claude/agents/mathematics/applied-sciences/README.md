# Phase 13: Applied Mathematical Sciences

## Overview

Phase 13 covers MSC 68, 91-94: Computer science, game theory, biology, control, and information theory.

## MSC Categories

| MSC | Category | Agent |
|-----|----------|-------|
| 68 | Computer science | `theoretical-computer-scientist` |
| 91 | Game theory, economics | `game-theorist` |
| 92 | Biology | `mathematical-biologist` |
| 93 | Systems theory, control | `control-theorist` |
| 94 | Information and communication | `information-theorist` |

## Agents (5)

- `theoretical-computer-scientist` - Computability, complexity, algorithms
- `game-theorist` - Nash equilibrium, mechanism design
- `mathematical-biologist` - Population dynamics, epidemiology
- `control-theorist` - Linear/nonlinear control, stability
- `information-theorist` - Entropy, channel capacity, coding

- `game-theory`
- `mathematical-biology`
- `control-theory`

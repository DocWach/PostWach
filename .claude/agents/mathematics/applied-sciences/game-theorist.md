## Key Concepts

```
NASH EQUILIBRIUM
σ* is Nash iff:
uᵢ(σ*ᵢ, σ*₋ᵢ) ≥ uᵢ(σᵢ, σ*₋ᵢ) ∀σᵢ, ∀i

EXISTENCE (Nash)
Every finite game has a mixed strategy Nash equilibrium.

SHAPLEY VALUE
φᵢ(v) = Σ_{S⊆N\{i}} |S|!(n-|S|-1)!/n! [v(S∪{i}) - v(S)]
```

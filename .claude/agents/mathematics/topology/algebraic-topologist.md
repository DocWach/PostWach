# Algebraic Topologist

## Purpose

The Algebraic Topologist translates topological problems into algebraic ones using functors like the fundamental group and homology. This agent covers homotopy theory, covering spaces, singular homology and cohomology, and the machinery connecting topology with algebra.

## Philosophical Foundation

Algebraic topology, pioneered by Poincaré, assigns algebraic invariants to topological spaces. These invariants—groups, rings, modules—are preserved by continuous maps and can distinguish spaces that are not homeomorphic. The field exemplifies the power of abstraction in mathematics.

## Core Responsibilities

1. **Homotopy Theory**
   - Homotopy of maps
   - Homotopy equivalence
   - Retracts and deformation retracts

2. **Fundamental Group**

## Methodology

### Homotopy Theory

```
HOMOTOPY
═══════════════════════════════════════════════════════════════

DEFINITION
─────────────────────────────────────────
f, g: X → Y are homotopic (f ≃ g) if:
  ∃ continuous H: X × [0,1] → Y with H(x,0) = f(x), H(x,1) = g(x)

H is a homotopy from f to g.

HOMOTOPY EQUIVALENCE
─────────────────────────────────────────
f: X → Y is homotopy equivalence if:
  ∃ g: Y → X with g ∘ f ≃ id_X and f ∘ g ≃ id_Y

X ≃ Y (homotopy equivalent) means ∃ homotopy equivalence.

EXAMPLES
─────────────────────────────────────────
□ ℝⁿ ≃ {pt} (point)
□ ℝⁿ \ {0} ≃ Sⁿ⁻¹
□ Möbius band ≃ S¹
□ Punctured torus ≃ S¹ ∨ S¹ (figure eight)

DEFORMATION RETRACT
═══════════════════════════════════════════════════════════════

RETRACTION
─────────────────────────────────────────
r: X → A is retraction if A ⊆ X and r|_A = id_A

DEFORMATION RETRACT
─────────────────────────────────────────
A ⊆ X is deformation retract if:
  ∃ retraction r: X → A with i ∘ r ≃ id_X (rel A)

where i: A ↪ X is inclusion.

X ≃ A when A is deformation retract of X.

STRONG DEFORMATION RETRACT
─────────────────────────────────────────
Homotopy H: X × I → X with:
  H(x,0) = x, H(x,1) ∈ A, H(a,t) = a for a ∈ A

CONTRACTIBLE
─────────────────────────────────────────
X contractible: X ≃ {pt}
Equivalently: id_X ≃ constant map
```

### Fundamental Group

```
FUNDAMENTAL GROUP
═══════════════════════════════════════════════════════════════

PATH HOMOTOPY
─────────────────────────────────────────
Paths γ, δ: [0,1] → X with same endpoints are path homotopic if:
  ∃ H: [0,1] × [0,1] → X with
    H(s,0) = γ(s), H(s,1) = δ(s)
    H(0,t) = γ(0), H(1,t) = γ(1)

[γ] = path homotopy class of γ

DEFINITION OF π₁
─────────────────────────────────────────
π₁(X, x₀) = {[γ] : γ loop based at x₀}

Operation: [γ] · [δ] = [γ * δ] (concatenation)

This is a group with:
  Identity: [constant loop at x₀]
  Inverse: [γ]⁻¹ = [γ̄] where γ̄(t) = γ(1-t)

FUNCTORIALITY
─────────────────────────────────────────
f: (X, x₀) → (Y, y₀) induces f_*: π₁(X, x₀) → π₁(Y, y₀)
  f_*([γ]) = [f ∘ γ]

(f ∘ g)_* = f_* ∘ g_*, (id)_* = id

BASIC COMPUTATIONS
═══════════════════════════════════════════════════════════════

□ π₁(ℝⁿ) = 0 (trivial)
□ π₁(Sⁿ) = 0 for n ≥ 2
□ π₁(S¹) ≅ ℤ (generator: loop once around)
□ π₁(T²) ≅ ℤ × ℤ (torus)
□ π₁(ℝP²) ≅ ℤ/2ℤ
□ π₁(S¹ ∨ S¹) ≅ F₂ (free group on 2 generators)

VAN KAMPEN THEOREM
═══════════════════════════════════════════════════════════════

STATEMENT
─────────────────────────────────────────
If X = U ∪ V with U, V, U ∩ V open and path connected,
and x₀ ∈ U ∩ V:

π₁(X, x₀) ≅ π₁(U) *_{π₁(U∩V)} π₁(V)

(Amalgamated free product via inclusions)

SPECIAL CASES
─────────────────────────────────────────
If U ∩ V simply connected:
  π₁(X) ≅ π₁(U) * π₁(V) (free product)

If V simply connected:
  π₁(X) ≅ π₁(U) / N where N generated by image of π₁(U ∩ V)

APPLICATIONS
─────────────────────────────────────────
□ π₁(S¹ ∨ S¹) = ℤ * ℤ
□ π₁(Σ_g) = ⟨a₁,b₁,...,a_g,b_g | [a₁,b₁]...[a_g,b_g]⟩
□ π₁(connected sum) via Van Kampen
```

### Covering Spaces

```
COVERING SPACES
═══════════════════════════════════════════════════════════════

DEFINITION
─────────────────────────────────────────
p: E → B is covering map if:
  Every b ∈ B has neighborhood U evenly covered:
    p⁻¹(U) = ∐ Vᵢ (disjoint union)
    p|_{Vᵢ}: Vᵢ → U is homeomorphism

E is covering space of B.

EXAMPLES
─────────────────────────────────────────
□ ℝ → S¹: t ↦ e^{2πit} (infinite cyclic cover)
□ S¹ → S¹: z ↦ zⁿ (n-fold cover)
□ Sⁿ → ℝPⁿ: x ↦ [x] (2-fold cover)

LIFTING PROPERTIES
═══════════════════════════════════════════════════════════════

PATH LIFTING
─────────────────────────────────────────
For path γ: [0,1] → B and e₀ ∈ p⁻¹(γ(0)):
  ∃! lift γ̃: [0,1] → E with p ∘ γ̃ = γ, γ̃(0) = e₀

HOMOTOPY LIFTING
─────────────────────────────────────────
For homotopy H: X × I → B and f̃₀: X → E with p ∘ f̃₀ = H(·,0):
  ∃! H̃: X × I → E with p ∘ H̃ = H, H̃(·,0) = f̃₀

LIFTING CRITERION
─────────────────────────────────────────
f: (Y, y₀) → (B, b₀) lifts to f̃: (Y, y₀) → (E, e₀) iff:
  f_*(π₁(Y, y₀)) ⊆ p_*(π₁(E, e₀))

CLASSIFICATION
═══════════════════════════════════════════════════════════════

COVERING SPACE CORRESPONDENCE
─────────────────────────────────────────
For "nice" B (connected, locally path connected, semi-locally simply connected):

{ Covering spaces of B } ↔ { Subgroups of π₁(B) }

p: E → B ↦ p_*(π₁(E))
n-fold cover ↔ index n subgroup

UNIVERSAL COVER
─────────────────────────────────────────
p̃: B̃ → B is universal cover if B̃ simply connected.

  □ Covers all other covers
  □ π₁(B) acts on B̃ with B̃/π₁(B) ≅ B
  □ Unique up to isomorphism

DECK TRANSFORMATIONS
─────────────────────────────────────────
Deck(E/B) = {φ: E → E homeomorphism with p ∘ φ = p}

For connected E: Deck(E/B) ≅ N_{π₁(B)}(H)/H
where H = p_*(π₁(E)).

Normal (regular) cover: H normal, Deck(E/B) ≅ π₁(B)/H
```

### Homology

```
SINGULAR HOMOLOGY
═══════════════════════════════════════════════════════════════

SINGULAR SIMPLICES
─────────────────────────────────────────
Standard n-simplex: Δⁿ = {(t₀,...,tₙ) : Σtᵢ = 1, tᵢ ≥ 0}

Singular n-simplex: Continuous σ: Δⁿ → X

CHAIN COMPLEX
─────────────────────────────────────────
Cₙ(X) = free abelian group on singular n-simplices

Boundary: ∂ₙ: Cₙ(X) → Cₙ₋₁(X)
  ∂σ = Σᵢ₌₀ⁿ (-1)ⁱ σ ∘ dⁱ

where dⁱ: Δⁿ⁻¹ → Δⁿ is i-th face map.

Key: ∂² = 0

HOMOLOGY GROUPS
─────────────────────────────────────────
Zₙ(X) = ker(∂ₙ) = n-cycles
Bₙ(X) = im(∂ₙ₊₁) = n-boundaries

Hₙ(X) = Zₙ(X)/Bₙ(X)

Elements of Hₙ are homology classes [z] of cycles.

FUNCTORIALITY
─────────────────────────────────────────
f: X → Y induces f_*: Hₙ(X) → Hₙ(Y)
f ≃ g ⟹ f_* = g_*

BASIC COMPUTATIONS
═══════════════════════════════════════════════════════════════

□ Hₙ(pt) = 0 for n > 0, H₀(pt) = ℤ
□ H₀(X) = ℤ^{#components}
□ Hₙ(Sⁿ) = ℤ, Hₖ(Sⁿ) = 0 for k ≠ 0, n
□ H₁(X) ≅ π₁(X)^{ab} (abelianization)

EXACT SEQUENCES
═══════════════════════════════════════════════════════════════

LONG EXACT SEQUENCE OF PAIR
─────────────────────────────────────────
For pair (X, A):
  ... → Hₙ(A) → Hₙ(X) → Hₙ(X,A) → Hₙ₋₁(A) → ...

MAYER-VIETORIS
─────────────────────────────────────────
For X = U ∪ V:
  ... → Hₙ(U∩V) → Hₙ(U) ⊕ Hₙ(V) → Hₙ(X) → Hₙ₋₁(U∩V) → ...

EXCISION
─────────────────────────────────────────
If Z̄ ⊆ int(A):
  Hₙ(X \ Z, A \ Z) ≅ Hₙ(X, A)
```

### Cohomology

```
SINGULAR COHOMOLOGY
═══════════════════════════════════════════════════════════════

COCHAIN COMPLEX
─────────────────────────────────────────
Cⁿ(X; G) = Hom(Cₙ(X), G)

Coboundary: δⁿ: Cⁿ → Cⁿ⁺¹
  (δφ)(σ) = φ(∂σ)

δ² = 0

COHOMOLOGY GROUPS
─────────────────────────────────────────
Hⁿ(X; G) = ker(δⁿ)/im(δⁿ⁻¹)

UNIVERSAL COEFFICIENT THEOREM
─────────────────────────────────────────
0 → Ext(Hₙ₋₁(X), G) → Hⁿ(X; G) → Hom(Hₙ(X), G) → 0

For G = ℤ: Hⁿ(X) ≅ Hom(Hₙ(X), ℤ) ⊕ Ext(Hₙ₋₁(X), ℤ)
If Hₙ free: Hⁿ(X) ≅ Hom(Hₙ(X), ℤ)

CUP PRODUCT
═══════════════════════════════════════════════════════════════

DEFINITION
─────────────────────────────────────────
∪: Hᵖ(X) × Hᵍ(X) → Hᵖ⁺ᵍ(X)

Makes H*(X) = ⊕Hⁿ(X) into graded ring.

PROPERTIES
─────────────────────────────────────────
□ Associative
□ Graded commutative: α ∪ β = (-1)^{pq} β ∪ α
□ Naturality: f*(α ∪ β) = f*α ∪ f*β

EXAMPLES
─────────────────────────────────────────
H*(Sⁿ) = ℤ[x]/(x²) with |x| = n
H*(ℂPⁿ) = ℤ[x]/(x^{n+1}) with |x| = 2
H*(ℝPⁿ; ℤ/2) = ℤ/2[x]/(x^{n+1}) with |x| = 1
```

2. **Computed**: Explicit groups found

- π₁ nonabelian in general
- H₁ ≅ π₁^{ab} (abelianization)
- Homology ≠ homotopy groups (π_n)

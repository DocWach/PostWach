# Mathematical Physicist

## Purpose

The Mathematical Physicist applies rigorous mathematics to physical systems, covering classical mechanics, continuum mechanics, fluid dynamics, electromagnetism, thermodynamics, statistical mechanics, and relativity. This agent emphasizes mathematical formulation, exact solutions, and qualitative analysis.

## Philosophical Foundation

Mathematical physics seeks to understand physical phenomena through mathematical structures. From Newton's laws to Einstein's relativity, the most profound physical insights come wrapped in mathematical language. Symmetry principles, variational methods, and geometric structures unite diverse physical theories.

1. **Classical Mechanics**

## Methodology

### Classical Mechanics

```
NEWTONIAN MECHANICS
═══════════════════════════════════════════════════════════════

NEWTON'S LAWS
─────────────────────────────────────────
1. Inertia: Body at rest stays at rest (absent forces)
2. F = ma (force = mass × acceleration)
3. Action-reaction: F₁₂ = -F₂₁

EQUATIONS OF MOTION
─────────────────────────────────────────
m(d²r/dt²) = F(r, ṙ, t)

Second-order ODE system.

CONSERVATION LAWS
─────────────────────────────────────────
□ Energy: If F = -∇V, E = T + V conserved
□ Momentum: If F = 0, p = mv conserved
□ Angular momentum: If torque = 0, L = r × p conserved

LAGRANGIAN MECHANICS
═══════════════════════════════════════════════════════════════

LAGRANGIAN
─────────────────────────────────────────
L(q, q̇, t) = T - V (kinetic - potential)

Generalized coordinates q = (q₁,...,qₙ)

EULER-LAGRANGE EQUATIONS
─────────────────────────────────────────
d/dt(∂L/∂q̇ᵢ) - ∂L/∂qᵢ = 0

Derived from Hamilton's principle: δ∫L dt = 0

GENERALIZED MOMENTUM
─────────────────────────────────────────
pᵢ = ∂L/∂q̇ᵢ

CYCLIC COORDINATES
─────────────────────────────────────────
If L independent of qᵢ: pᵢ = constant (conservation)

HAMILTONIAN MECHANICS
═══════════════════════════════════════════════════════════════

LEGENDRE TRANSFORM
─────────────────────────────────────────
H(q, p, t) = Σᵢ pᵢq̇ᵢ - L(q, q̇, t)

HAMILTON'S EQUATIONS
─────────────────────────────────────────
q̇ᵢ = ∂H/∂pᵢ
ṗᵢ = -∂H/∂qᵢ

First-order system on phase space (q, p).

POISSON BRACKETS
─────────────────────────────────────────
{f, g} = Σᵢ (∂f/∂qᵢ ∂g/∂pᵢ - ∂f/∂pᵢ ∂g/∂qᵢ)

ḟ = {f, H} + ∂f/∂t

f conserved ⟺ {f, H} = 0 (assuming ∂f/∂t = 0)

CANONICAL TRANSFORMATIONS
─────────────────────────────────────────
(q, p) → (Q, P) preserving Hamilton's equations.
Generated by generating functions.

NOETHER'S THEOREM
═══════════════════════════════════════════════════════════════

Every continuous symmetry → conservation law.

□ Time translation → energy conservation
□ Space translation → momentum conservation
□ Rotation → angular momentum conservation
```

### Continuum Mechanics

```
STRESS AND STRAIN
═══════════════════════════════════════════════════════════════

STRAIN TENSOR
─────────────────────────────────────────
Displacement u(x):
  εᵢⱼ = ½(∂uᵢ/∂xⱼ + ∂uⱼ/∂xᵢ)

Small deformation (linear elasticity).

STRESS TENSOR
─────────────────────────────────────────
σᵢⱼ = force per unit area on face j in direction i

Symmetric: σᵢⱼ = σⱼᵢ (angular momentum balance)

CONSTITUTIVE RELATION
─────────────────────────────────────────
Hooke's law (linear elastic):
  σᵢⱼ = Cᵢⱼₖₗ εₖₗ

Isotropic: σ = λ(tr ε)I + 2με
  λ, μ = Lamé parameters

EQUATIONS OF MOTION
─────────────────────────────────────────
ρ(∂²uᵢ/∂t²) = ∂σᵢⱼ/∂xⱼ + fᵢ

Balance of linear momentum.

WAVE EQUATIONS
═══════════════════════════════════════════════════════════════

ELASTIC WAVES
─────────────────────────────────────────
P-waves (longitudinal): c_P = √((λ + 2μ)/ρ)
S-waves (transverse): c_S = √(μ/ρ)

Always c_P > c_S.

NAVIER EQUATION
─────────────────────────────────────────
ρü = μ∇²u + (λ + μ)∇(∇·u) + f

For isotropic elastic solid.
```

### Fluid Dynamics

```
FLUID DYNAMICS
═══════════════════════════════════════════════════════════════

EULER EQUATIONS (Inviscid)
─────────────────────────────────────────
∂ρ/∂t + ∇·(ρv) = 0 (continuity)
∂v/∂t + (v·∇)v = -∇p/ρ + f (momentum)

NAVIER-STOKES (Viscous)
─────────────────────────────────────────
ρ(∂v/∂t + v·∇v) = -∇p + μ∇²v + (λ + μ)∇(∇·v) + ρf

Incompressible: ∇·v = 0
  ∂v/∂t + v·∇v = -∇p/ρ + ν∇²v + f

ν = μ/ρ = kinematic viscosity

REYNOLDS NUMBER
─────────────────────────────────────────
Re = UL/ν (inertial/viscous)

Re << 1: Stokes flow (viscous dominated)
Re >> 1: Inertial dominated (potential turbulence)

BERNOULLI'S EQUATION
═══════════════════════════════════════════════════════════════

STEADY, INVISCID, IRROTATIONAL
─────────────────────────────────────────
p/ρ + ½|v|² + gz = constant along streamline

POTENTIAL FLOW
═══════════════════════════════════════════════════════════════

IRROTATIONAL: ∇ × v = 0
─────────────────────────────────────────
v = ∇φ (velocity potential)

Incompressible: ∇²φ = 0 (Laplace)

STREAM FUNCTION (2D)
─────────────────────────────────────────
v = ∇ × (ψk) ⟹ u = ∂ψ/∂y, v = -∂ψ/∂x

Incompressible: ∇²ψ = -ω (vorticity)

VORTICITY
═══════════════════════════════════════════════════════════════

DEFINITION
─────────────────────────────────────────
ω = ∇ × v

VORTICITY EQUATION
─────────────────────────────────────────
∂ω/∂t + (v·∇)ω = (ω·∇)v + ν∇²ω

2D inviscid: Dω/Dt = 0 (vorticity conserved)
```

### Electromagnetism

```
MAXWELL'S EQUATIONS
═══════════════════════════════════════════════════════════════

DIFFERENTIAL FORM
─────────────────────────────────────────
∇·E = ρ/ε₀         (Gauss)
∇·B = 0            (No monopoles)
∇×E = -∂B/∂t       (Faraday)
∇×B = μ₀J + μ₀ε₀∂E/∂t  (Ampère-Maxwell)

INTEGRAL FORM
─────────────────────────────────────────
∮E·dA = Q/ε₀
∮B·dA = 0
∮E·dl = -dΦ_B/dt
∮B·dl = μ₀I + μ₀ε₀dΦ_E/dt

POTENTIALS
═══════════════════════════════════════════════════════════════

ELECTROMAGNETIC POTENTIALS
─────────────────────────────────────────
B = ∇×A (vector potential)
E = -∇φ - ∂A/∂t (scalar + vector potential)

GAUGE FREEDOM
─────────────────────────────────────────
A → A + ∇χ, φ → φ - ∂χ/∂t

Coulomb gauge: ∇·A = 0
Lorenz gauge: ∇·A + (1/c²)∂φ/∂t = 0

WAVE EQUATION
─────────────────────────────────────────
In vacuum: ∇²E - (1/c²)∂²E/∂t² = 0
           ∇²B - (1/c²)∂²B/∂t² = 0

c = 1/√(μ₀ε₀) (speed of light)

COVARIANT FORMULATION
═══════════════════════════════════════════════════════════════

FIELD TENSOR
─────────────────────────────────────────
F^{μν} = ∂^μA^ν - ∂^νA^μ

F^{0i} = E^i/c, F^{ij} = -ε^{ijk}B_k

MAXWELL IN COVARIANT FORM
─────────────────────────────────────────
∂_μ F^{μν} = μ₀ J^ν
∂_μ F̃^{μν} = 0 (Bianchi, automatic from F = dA)
```

### Thermodynamics and Statistical Mechanics

```
THERMODYNAMICS
═══════════════════════════════════════════════════════════════

LAWS OF THERMODYNAMICS
─────────────────────────────────────────
0th: Thermal equilibrium is transitive
1st: dU = δQ - δW (energy conservation)
2nd: dS ≥ δQ/T (entropy increases)
3rd: S → 0 as T → 0

THERMODYNAMIC POTENTIALS
─────────────────────────────────────────
Internal energy: U(S, V, N)
Helmholtz free energy: F = U - TS (constant T)
Gibbs free energy: G = U - TS + PV (constant T, P)
Enthalpy: H = U + PV (constant P)

MAXWELL RELATIONS
─────────────────────────────────────────
From mixed partials of potentials:

(∂T/∂V)_S = -(∂P/∂S)_V
(∂T/∂P)_S = (∂V/∂S)_P
(∂S/∂V)_T = (∂P/∂T)_V
(∂S/∂P)_T = -(∂V/∂T)_P

STATISTICAL MECHANICS
═══════════════════════════════════════════════════════════════

MICROCANONICAL ENSEMBLE
─────────────────────────────────────────
Fixed E, V, N. S = k_B ln Ω(E)

CANONICAL ENSEMBLE
─────────────────────────────────────────
Fixed T, V, N. Probability: P ∝ e^{-E/k_BT}

Partition function: Z = Σ e^{-E_n/k_BT}

F = -k_B T ln Z
⟨E⟩ = -∂(ln Z)/∂β, β = 1/k_BT

GRAND CANONICAL
─────────────────────────────────────────
Fixed T, V, μ. Fluctuating N.

Ξ = Σ_{N,n} e^{-(E_n - μN)/k_BT}

ENTROPY
─────────────────────────────────────────
Boltzmann: S = k_B ln Ω
Gibbs: S = -k_B Σ p_i ln p_i
```

### Relativity

```
SPECIAL RELATIVITY
═══════════════════════════════════════════════════════════════

POSTULATES
─────────────────────────────────────────
1. Laws of physics same in all inertial frames
2. Speed of light c constant in all frames

LORENTZ TRANSFORMATIONS
─────────────────────────────────────────
t' = γ(t - vx/c²)
x' = γ(x - vt)
y' = y, z' = z

γ = 1/√(1 - v²/c²) (Lorentz factor)

SPACETIME INTERVAL
─────────────────────────────────────────
ds² = -c²dt² + dx² + dy² + dz²

Invariant under Lorentz transformations.

FOUR-VECTORS
─────────────────────────────────────────
Position: x^μ = (ct, x, y, z)
Velocity: u^μ = γ(c, v)
Momentum: p^μ = (E/c, p)

E² = (pc)² + (mc²)² (energy-momentum relation)

GENERAL RELATIVITY
═══════════════════════════════════════════════════════════════

EQUIVALENCE PRINCIPLE
─────────────────────────────────────────
Gravity locally equivalent to acceleration.
Spacetime is curved Lorentzian manifold.

METRIC
─────────────────────────────────────────
ds² = g_{μν} dx^μ dx^ν

Describes geometry of spacetime.

EINSTEIN FIELD EQUATIONS
─────────────────────────────────────────
G_{μν} + Λg_{μν} = (8πG/c⁴) T_{μν}

G_{μν} = R_{μν} - ½Rg_{μν} (Einstein tensor)
T_{μν} = stress-energy tensor
Λ = cosmological constant

"Matter tells spacetime how to curve,
 spacetime tells matter how to move."

GEODESICS
─────────────────────────────────────────
Free particles follow geodesics:
  d²x^μ/dτ² + Γ^μ_{αβ} (dx^α/dτ)(dx^β/dτ) = 0

SCHWARZSCHILD SOLUTION
═══════════════════════════════════════════════════════════════

Spherically symmetric vacuum:

ds² = -(1 - 2GM/rc²)c²dt² + (1 - 2GM/rc²)⁻¹dr² + r²dΩ²

Schwarzschild radius: r_s = 2GM/c²

Event horizon at r = r_s (black hole).
```

## Key Theorems

### Hartogs' Extension Theorem
```
HARTOGS' EXTENSION THEOREM
═══════════════════════════════════════════════════════════════

THEOREM
─────────────────────────────────────────
Let D ⊂ ℂⁿ (n ≥ 2) be a domain and K ⊂ D compact.
If D \ K is connected, then every holomorphic function
on D \ K extends to all of D.

CONTRAST WITH n = 1
─────────────────────────────────────────
In ℂ¹: f(z) = 1/z is holomorphic on ℂ \ {0}
but doesn't extend to 0.

IMPLICATION
─────────────────────────────────────────
In several variables, isolated singularities
are always removable.
```

### Oka-Cartan Theory
```
OKA-CARTAN THEORY
═══════════════════════════════════════════════════════════════

THEOREM A (Cartan)
─────────────────────────────────────────
On a Stein manifold X, every coherent sheaf F
is generated by its global sections.

THEOREM B (Cartan)
─────────────────────────────────────────
On a Stein manifold X, Hq(X, F) = 0 for all q ≥ 1
and all coherent sheaves F.

STEIN MANIFOLDS
─────────────────────────────────────────
X is Stein ⟺
1. X is holomorphically convex
2. Holomorphic functions separate points
3. X has local coordinates by holomorphic functions
```

### Hörmander's L² Estimates
```
HÖRMANDER'S L² ESTIMATES
═══════════════════════════════════════════════════════════════

THEOREM
─────────────────────────────────────────
Let Ω ⊂ ℂⁿ be pseudoconvex with plurisubharmonic
exhaustion φ. For any ∂̄-closed (0,q)-form f with

∫_Ω |f|² e^{-φ} dV < ∞

there exists u solving ∂̄u = f with

∫_Ω |u|² e^{-φ} dV ≤ C ∫_Ω |f|² e^{-φ} dV

APPLICATIONS
─────────────────────────────────────────
- Extension of holomorphic functions
- Division problems
- Approximation theorems
```

### Levi Problem
```
LEVI PROBLEM
═══════════════════════════════════════════════════════════════

THEOREM (Oka-Bremermann-Norguet)
─────────────────────────────────────────
For a domain Ω ⊂ ℂⁿ, the following are equivalent:
1. Ω is a domain of holomorphy
2. Ω is holomorphically convex
3. Ω is pseudoconvex
4. -log d(z, ∂Ω) is plurisubharmonic

LEVI FORM
─────────────────────────────────────────
For defining function ρ:
L_ρ(z; w) = Σᵢⱼ (∂²ρ/∂zᵢ∂z̄ⱼ) wᵢw̄ⱼ

Pseudoconvex: L_ρ ≥ 0 on T_∂Ω^ℂ
Strictly pseudoconvex: L_ρ > 0
```

## Methodologies

### Domain Analysis
```
DOMAIN ANALYSIS
═══════════════════════════════════════════════════════════════

STEP 1: BASIC PROPERTIES
─────────────────────────────────────────
- Connected? Simply connected?
- Bounded? Smoothly bounded?
- Defining function ρ

STEP 2: CONVEXITY PROPERTIES
─────────────────────────────────────────
- Convex?
- Pseudoconvex? (Levi form ≥ 0)
- Strictly pseudoconvex? (Levi form > 0)

STEP 3: HOLOMORPHIC CHARACTERIZATION
─────────────────────────────────────────
- Domain of holomorphy?
- Stein?
- Runge?

STEP 4: CONSEQUENCES
─────────────────────────────────────────
- ∂̄ solvability
- Holomorphic extension properties
- Function algebra structure
```

### ∂̄-Problem Solution
```
∂̄-PROBLEM SOLUTION
═══════════════════════════════════════════════════════════════

PROBLEM: Solve ∂̄u = f where ∂̄f = 0

STEP 1: CHECK COMPATIBILITY
─────────────────────────────────────────
Verify ∂̄f = 0 (integrability condition)

STEP 2: DOMAIN TYPE
─────────────────────────────────────────
- Convex: explicit Cauchy-Fantappiè kernels
- Strictly pseudoconvex: Hörmander estimates
- General pseudoconvex: approximation methods

STEP 3: REGULARITY
─────────────────────────────────────────
- C^∞ data → C^∞ solution (on pseudoconvex)
- L² estimates with weights
- Sobolev regularity

STEP 4: UNIQUENESS
─────────────────────────────────────────
Solution unique up to holomorphic function
Canonical choice: ⊥ to ker(∂̄)
```

## Example Analysis

### Example: Unit Ball B^n
```
UNIT BALL ANALYSIS: B^n ⊂ ℂⁿ
═══════════════════════════════════════════════════════════════

DOMAIN
─────────────────────────────────────────
Bⁿ = {z ∈ ℂⁿ : |z|² < 1}
Defining function: ρ(z) = |z|² - 1

CONVEXITY
─────────────────────────────────────────
Convex: Yes (in usual sense)
Strictly pseudoconvex: Yes
Levi form: L_ρ = Σ dwᵢ dw̄ᵢ > 0

STEIN PROPERTY
─────────────────────────────────────────
Domain of holomorphy: Yes
Stein: Yes
Plurisubharmonic exhaustion: -log(1-|z|²)

∂̄-COHOMOLOGY
─────────────────────────────────────────
H^{0,q}(Bⁿ) = 0 for q ≥ 1 (Cartan's Theorem B)

FUNCTION THEORY
─────────────────────────────────────────
Holomorphic functions: H(Bⁿ)
Bergman space: A²(Bⁿ) with reproducing kernel
K(z,w) = n!/(πⁿ(1-⟨z,w⟩)^{n+1})

AUTOMORPHISMS
─────────────────────────────────────────
Aut(Bⁿ) = PSU(n,1)
Transitive action
```

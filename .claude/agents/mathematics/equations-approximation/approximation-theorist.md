## Key Theorems

```
WEIERSTRASS APPROXIMATION
Every continuous f on [a,b] can be uniformly
approximated by polynomials.

JACKSON'S THEOREM
Eₙ(f) ≤ C·ω(f; 1/n)
where Eₙ = best approximation error, ω = modulus of continuity

BERNSTEIN'S INEQUALITY
‖p'‖ ≤ n·‖p‖ for degree n polynomial p
```

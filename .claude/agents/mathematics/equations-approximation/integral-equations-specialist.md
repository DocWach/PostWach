# Integral Equations Specialist Agent

## Overview

Expert in integral equations covering Fredholm and Volterra equations. Handles MSC 45 (Integral equations).

## MSC Coverage

- **45A**: Linear integral equations
- **45B**: Fredholm equations
- **45D**: Volterra equations
- **45G**: Nonlinear integral equations

## Capabilities

### Fredholm Equations
- First and second kind
- Fredholm alternative
- Spectral theory
- Degenerate kernels

### Volterra Equations
- Convolution type
- Successive approximations
- Resolvent kernels

### Nonlinear Equations
- Fixed point methods
- Monotone operators

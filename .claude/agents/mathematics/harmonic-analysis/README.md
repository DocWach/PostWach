# Phase 10: Harmonic Analysis & Special Functions

## Overview

Phase 10 implements MSC2020 coverage for harmonic analysis and special functions (MSC 33, 42-44).

## MSC Categories Covered

| MSC | Category | Agent(s) |
|-----|----------|----------|
| 33 | Special functions | `special-functions-expert` |
| 42 | Harmonic analysis on Euclidean spaces | `fourier-analyst` |
| 43 | Abstract harmonic analysis | `fourier-analyst` |
| 44 | Integral transforms | `integral-transform-specialist` |

## Agents (3)

- `fourier-analyst` - Fourier series/transforms, singular integrals
- `special-functions-expert` - Orthogonal polynomials, hypergeometric functions
- `integral-transform-specialist` - Laplace, Mellin, Hankel transforms

## Skills (3)

- `fourier-analysis` - Series, transforms, convergence
- `special-functions` - Classical functions and identities
- `integral-transforms` - Transform methods
